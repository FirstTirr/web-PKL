Dashboard

Gunakan sidebar di sebelah kiri untuk mengelola akun dan perusahaan. Sidebar ini mendukung operasi CRUD lengkap untuk kedua entitas.

Manajemen Akun

Kelola akun pengguna dengan fitur:

  • Tambah akun baru
  • Lihat semua akun
  • Edit informasi akun
  • Hapus akun

Manajemen Perusahaan

Kelola data perusahaan dengan fitur:

  • Tambah perusahaan baru
  • Lihat semua perusahaan
  • Edit informasi perusahaan
  • Hapus perusahaan
